高三复习备考- 小题突破——数学文化

Author: laoluokj Date: October 9, 2020  14:19:59

摘要

学生2019届文科考生，课题为《高三二轮专题复习——数学文化》总结课，此前经历了一周的专题复习，积累了一些解此类题的经验，但是没有概括和梳理，没有进一步反思和升华。总结课希望利用两个题组以及两道综合例题，将多个知识点呈现出来，有机的结合众多知识点，达到由点出发，形成线面的作用，多角度，多视角的方式看待数学例题，感悟数学文化。
希望传达的核心思想是：数学文化有显性和隐性两部分组成，显性是隐形的载体。课堂中不断回顾，不断反思，不断试错纠错，慢慢推进，慢慢提升的过程是数学文化真正的体现，是数学思想、数学精神、数学意识和数学观念的体现。

数学文化，学习策略，题组训练

设计思路

习题教学是课堂教学不可忽视的环节，著名数学家G·波利亚曾经说过：“一个专心的认真备课的教师，能够拿出一个有意义的但又不太复杂的题目，去帮助学生发掘问题的各个方面，使得通过这道题，就好像通过一扇门，把学生引入一个完整的理论领域。”通过设置题组和经典例题巩固新知，发展能力，在提高学生数学素养。

注重学生的展示，表现出对知识的掌握情况和容易出现错误的关键点。将学生放置在主体地位，老师成为一个旁观者，观察学生的行为特征和察觉学生思考的特点，在出现问题和出现分歧时聆听学生的心声，交流个人心得。尊重学生的独立人格，放大学生的人格魅力，培养其自信心和拼搏斗志。

数学是每一个学生求学必须要学习的主课，它教给孩子们的不应只是冰冷的数学知识，更重要是要教给学生用数学的眼光看待问题、用数学的思想去思考问题，数学文化具有自己独特的魅力，理解和感悟数学思想、数学精神、数学意识和数学观念对个人发展尤为重要。

教学目标

  1. 知识与技能：多个高考高频考察知识点
  2. 过程与方法：
    1. 题组引导，大题收官
    2. 由浅入深，层层推进
    3. 反思感悟，内化吸收
  3. 情感态度价值观：
    1. 用数学的观点观察现实，构造数学模型，学习数学的语言、图表、符号表示，进行数学交流。
    2. 通过理性思维，培养严谨素质，追求创新精神，欣赏数学之美。

教学重点难点

教学重点：梳理清楚数学文化类试题的解答步骤和回顾多个高中数学必考知识点

教学难点：引导学生利用题组和例题将众多知识点融汇贯通，构建起较为完整的知识体系。

教学过程

新课引入

交流个人对数学文化复习的认识

数学文化专题复习比较困难，复习内容编排难度较大，很可能出现出力不讨好的状况。真实的考试中，学生肯定会面临新的试题，这代表着备考中完成的所有例题都不会完封不动的出现试卷中，需要学生能够真正理解“数学文化”甚至理解“文化”这一类难以解释说明的字眼。

它要求我们解决的不是习题，而是问题，重点强调了学以致用，灵活变通，触类旁通的能力，能力不是一朝一夕形成的，需要长期的积累和领悟，以及规律的训练和巩固。

它往往是一套试卷中的亮点，天然具有“网红”潜质，具有文化深度，全新视角，创新精神，同时它又琢磨不透，摸不到脉络。

  1. 表达老师个人一点见解，指导学生认识数学文化复习策略
  2. 希望能力尚可的学生能够向更加有深度的问题发起挑战，从提高能力的角度入手，组织和安排自己的复习内容
  3. 希望能力不足的学生能够量力而行，从高考整体成功的角度出发，明确取舍得失，有的放矢，高效备考

展示2019年高考全国卷

  1. “考了朵云？”
  2. “断臂维纳斯有多高？”
  3. “你的草稿纸用完了吗？”

探讨人体黄金分割之美,将美育教育融入数学教育,考查类比归纳与合情推理,渗透了逻辑推理和数学运算的核心素养。让维纳斯进入了高考数学,可以说这是一次全国性的数学美的普及活动,使人们对抽象的数学不得不刮目相看。

  1. 理解数学文化类型试题在高考中体现的意义
  2. 感受数学与其他学科相互渗透的关系
  3. 复习回顾已经精细讲解的经典试题
  4. 提醒学生不能怕，也不可怕，可以通过必要的反思和总结，找到其中的关键要素，只要积极的思考，深入的探索，可以构建出应对此类试题的方法论和策略集

第一组基础知识回顾

请四位同学上台，兵分四路为我们解决简单的例题。

老师及其他同学观察演示同学的完成情况，协助他们一起完成例题。

{二倍角公式}

已知函数 $y=(\sin x + \cos x)^2 + 2\cos^{2} x$,求函数的递减区间？

  1. 展现二倍角公式常考的方式方法
  2. 复习三角函数的基本方法基本技能
  3. 发现学生存在的问题

{简单几何体的表面积和体积公式}

  1. 强调公式记忆的重要性
  2. 强调公式使用的状况
  3. 提醒同学主动记忆比较复杂的台体计算公式

{扇形面积}

在半径为$6 cm$的圆中,圆心角为 $120^{\circ}$的扇形的面积是多少$cm^{2}$.

  1. 简单数据回顾扇形面积计算公式
  2. 对比真实计算状况和理想化计算状况在计算难度上的差别，提醒此时的考点蕴含对数学运算核心素养的考察

{写出常用的几组勾股数}

  1. 利用记忆的技巧记忆常见的勾股数
  2. 提高对数字的敏感性，当看到与记忆内容有关的数据时，可以预估计算的难以程度，进一步明确步骤和方向
  3. 利用记忆提高运算的速度，为考试争取宝贵时间

{综合练习}

(注：1丈=10尺=100寸，$\pi \thickapprox 3.14,\sin 22.5^{^\circ} \thickapprox \frac{5}{13}$)

第二组基础知识回顾

请一位同学在黑板解答例题。

老师检查基础知识掌握情况，回顾对应方法使用的条件和提示变通的想法。

{基本不等式}

若 $x>0,y>0$,且 $x+y=18$,则 $\sqrt{xy}$的最大值为？

A: $9$
B: $18$
C: $36$
D: $81$

{圆锥曲线运算}

已知椭圆C $\frac{y^2}{a^2} + \frac{x^2}{b^2} = 1 (a>b>0)$经过点 $\frac{3}{2},1$,一个焦点是 $F(0,1)$.求椭圆C的方程.

  1. 设置铺垫，让学生完成熟悉的内容，为第二道大题架设梯子，帮助学生发现最关键的问题。
  2. 复习回顾常考知识点，引导学生发现本质问题。
  3. 注意试题中设置的阅读陷阱。
  4. 尝试使用猜想和验证的方法跳过步骤繁琐的运算。
  5. 体现出的知识点：基本不等式，解析几何计算。

第二道数学文化综合例题

请一位同学在黑板解答例题。

老师解读学生完成状况，帮助其发现问题，突破核心考点。

在古希腊数学家海伦的著作《测地术》中记载了著名的海伦公式,利用三角形的三边长求三角形的面积,若三角形的三边分别为 $a,b,c$,则其面积 $S = \sqrt{p(p-a)(p-b)(p-c)}$,这里 $p = \frac{a+b+c}{2}$.已知在 $\angle ABC$中,$BC=6,AB=2AC$,则当 $ABC$的面积最大时,$\sin A = ?$

  1. 考察了学生的化简整理运算的能力。
  2. 对比数形结合在解题中的差异。
  3. 链接前一位同学的解题步骤，变通使用相应的知识点，达到解题的目的，让学生感受到知识之间的关系，体会问题不同的呈现方式。
  4. 提示学生在解题是提前思考解题策略的问题，优先选择省时节力的解答方案
  5. 培养学生大胆猜测和小心求证的精神
  6. 体现出的知识点：余弦定理，基本不等式，换元法降次，二次函数求最值，建立之间坐标系解决几何问题，向量数量积，三角恒等式

课堂总结

课堂总结一_关于数学文化

我们对数学文化专题做了一个小小的总结，个人觉得不够，任何文化都不是学习获得，需要长时间的熏陶和积累，最终对你产生非常巨大的影响。

数学文化有显性和隐性两部分组成.

显性的数学文化是指数学概念、数学公式与定理、数学方法、数学语言、数学问题等知识性成分；隐性的数学文化是指数学思想、数学精神、数学意识和数学观念等观念性成分.

观念性成分一般隐含在知识性成分之中，知识性成分是观念性成分的载体，观念性成分随着知识性成分的承传而传播，随着知识性成分的发展而发展.将数学文化渗透到数学课堂教学中，不仅有助于学生对知识的理解， 提高学生的文化修养，还能让学生学到终身受益的数学精神、思想和方法.

不要小看现在积累的每一道试题，测试的每一份试卷，如果你认真的对待它，它会融入你的血脉，在你以后遇到问题时自然而然的流淌出来一些小小的，但是异常珍贵的灵感。

不要小看现在解决的每一个问题，书写的每一份答案，如何你认真的对待它，它会渗入你的思想，在你以后遇到麻烦时火石电光的迸发出来一些奇思妙想，帮你度过难关。

课堂总结三_关于高考备考

给大家介绍过丹尼尔·卡尼曼《思考的快与慢》，我们需要利用快和慢的连续切换达到省时节力和准确无误，帮助我们考场发挥出色，取得好成绩。

现在是不是可以提出一个”备考的多与少”。你们是高三的学生，还是一个月的复习时间，天气炎热，心里急躁，那么试卷没有做，那么多任务没有完成，心中要急，但是不要燥。

第三轮复习，需要把多变成少。怎么做到？

  1. 扎扎实实地完成老师讲解的重点习题
  2. 不要再停留在解题的层面，上升到解决问题的层面
  3. 熟悉的动作反复练习，变成自然的条件反射

教学反思

本课结束后教研组进行了评课议课活动，对此课进行了认真的剖析，提出众多中肯的意见。

  1. 第一个尖锐的问题是：数学文化体现在哪里？
  2. 第二个尖锐的问题是：这么多内容到底哪里是重点？

第一个问题反映了课程不完整的问题，课程设计如此，但是在执行中出现了被设计的没有提及，没有被设计的内容临时加入，显然由于时间问题，没有旗帜鲜明的提出自己对数学文化的认识，导致自己觉得已经讲过，讲授的内容反映了显性文化，讲解的过程，解答的过程体现隐形的文化，差画龙点睛的一笔，差深化拔高的一下，需要在语言的凝练和观点的准确方面继续努力。

教学设计是指导教学的工具，照本宣科往往会忽视现场同学的表现，忽视了学生的主体地位。个人认为反馈是课堂中最宝贵的东西，人和人之间在语言眼神动作情感上的交流是教学设计很难表现的内容，很多内容无法用文字和语言表达清楚，但是在与朝夕相处的同学之间却可以用其他的方式交流。笔者在多次课堂中感受到自己其实不是老师，自己其实是从课堂反馈中获得的学生。

第二个问题反映了课程理解的不同。我们的学生是长期处于学困处境的学生，题组中的基础练习是他们能够完成的内容，即便非常的简单，但是对他们很困难。综合性的例题从来都是跳过不做，连挑战的念头都没有。我认为他们差的恰恰是隐性的数学精神，数学意识，数学观念，希望他们抛弃简单例题的反复练习，将知识链接和知识贯通作为重点，转变对重点的认识，总结的习惯，自己领悟的习惯是重点，拆解组合的过程是重点，慢下来，深入一些的理念是重点。重点不是解决一些题，而是理解具有怎样的思维方式才可以让我们解题。
